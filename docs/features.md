# Features

## Current feature set

- File tree with lazy-loaded folders
- Tree sorting by `name`, `ext`, or `update`
- Multi-select, range select, rename, delete, copy, cut, paste
- File and folder creation
- Drag move inside the tree
- External drop into the tree
- Right-click context menus
- Multiple preview tabs
- Up to two panes
- Recent file chips
- Markdown rendering with outline and outline resize
- Text editing with save
- Editor search with `Cmd/Ctrl+F`
- Multi-selection editing with `Cmd/Ctrl+D`
- Undo / redo
- PDF, image, and CSV preview
- Launch Codex from Terminal.app
- Codex stats and usage tracking
- CPU / memory monitoring
- Recent files
- Settings for opacity, blur, models, usage divisors, reset schedule, and markdown heading styles
- Startup boot screen

## Notes

- CSV rendering is simple and row-limited.
- Non-PDF files larger than 5MB are not previewed.
- PDF and image previews are display-only.
